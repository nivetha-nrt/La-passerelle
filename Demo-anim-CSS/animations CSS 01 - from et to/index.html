Animation de la couleur d'un élément

Votre objectif est de parvenir à créer une animation qui fasse changer, en douceur, la couleur de fond du carré ci-dessous (une simple div).

Comment faire

Si vous ouvrez ce fichier dans VSCode, vous verrez qu'un style est directement intégré dans le "head", du document.

On aurait pu utiliser un fichier CSS distinct, mais pour un code ausi minimaliste, cela n'était pas nécessaire.

Vous allez devoir modifier le CSS - et uniquement le CSS - pour parvenir à créer une animation progressive. Celle-ci devra faire passer la couleur du carré de orange à red en 3s (3 secondes). Pour le moment, l'animation se fera automatiquement au lancement de la page.

Étape 1 : création de l'animation

Il faut déjà définir ce que l'on appelle les keyframes de l'animation. C'est à dire, les différentes étapes de celle-ci.
Dans notre cas, il va falloir définir un point de départ from pour lequel la couleur de fond de l'objet ciblé est orange.
Puis, une étape d'arrivée to, où la couleur de fond sera red.

Votre animation doit porter un nom unique, c'est vous qui le choisissez.

On écrit donc une règle CSS spéciale comme suit : 

                
                @keyframes nom_de_votre_animation {
                    from {
                        /* Vos règles CSS de départ */
                        /* C'est à dire l'apparence de départ du carré */
                    }
                    to {
                        /* Vos règles CSS d'arrivée */
                        /* C'est à dire l'apparence finale de votre élément */
                    }
                }

Attention !

Il s'agit bien d'une règle CSS à part entière, il ne faut pas essayer de mettre celle-ci à l'intérieur du CSS de votre div #animated_box.
On fait d'abord la règle CSS @keyframes, et ensuite, on fera la règle #animated_box.

Étape 2 : Attacher l'animation à la div #animated_box

Une fois définie votre règle CSS @keyframes, et les apparences de départ et d'arrivée de celle-ci, il ne reste qu'à appliquer cette animation à la div.

Pour cela, dans la règle CSS de #animated_box, vous devez ajouter :

                
                    animation-name: nom_de_votre_animation;
                    animation-duration: temps en secondes de l'animation;

Comme vous le voyez, c'est directement dans le CSS de la div que l'on dit quelle est l'animation à utiliser ( en utilisant le nom que l'on a créé précédement) et en combien de temps il faut jouer celle-ci.

À noter :

Le background-color de votre div est très important car c'est la couleur qui sera réellement affichée une fois l'animation terminée. En général, on utilise donc la même couleur que pour le to, de l'animation.

À retenir :

  • On crée une animation avec la règle @keyframe et on lui donne un nom unique.
  • On attache une animation à un élément grâce à la propriété animation-name.
  • Il est possible d'attacher plusieur fois la même animation à différents objects.
  • Il est possible de la faire durer plus ou moins longtemps un animation grâce à la propriété animation-duration.
