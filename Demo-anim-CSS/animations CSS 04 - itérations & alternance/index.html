Animation répétée

Refaites l'exercice précédent, mais cette fois-ci faites en sorte que votre animation se répète indéfiniment en ajoutant l'instruction animation-iteration-count:infinite; dans les règles de votre div.

Testez également l'instruction animation-direction:alternate; pour voir votre animation se jouer d'abord dans un sens, puis dans l'autre.

Aller plus loin !

Recommencez à jouer avec les positions et les couleurs en faisant cette fois boucler vos animations et varier l'alternance.

A la place de infinite, vous pouvez aussi préciser le nombre de fois que vous souhaitez que votre animation se reproduise.

Aller encore plus loin

Vous avez maintenant tout ce qu'il faut pour faire rebondir une balle à l'écran.

Essayez !

Rappel

Pour arrondir les bords de votre div utilisez la propriété border-radius.

Astuce

Pour positionner votre div en bas de son contenant, utilisez top: calc(100% - 200px).
C'est à dire, le calcul de 100% de la hauteur du contenant moins la hauteur de votre div.
