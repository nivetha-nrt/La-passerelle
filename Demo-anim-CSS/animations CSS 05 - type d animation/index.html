Types d'animation

Vous avez peut-être remarqué que par défaut les animations ne sont pas parfaitement linéaires.

Vous pouvez modifer cela en jouant avec la propriété animation-timing-function.

Les différentes valeurs possibles sont :

  • linear
  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • et quelques autres très spécifiques

Refaites une animation de mouvement de la box en modifiant cette propriété pour constater les différences dans l'animation.

Pour pouvoir facilement faire rentrer plusieur div les unes à côté des autres, la taille de la div a été réduite à 100px x 100px.
