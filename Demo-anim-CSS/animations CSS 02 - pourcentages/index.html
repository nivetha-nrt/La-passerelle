Animation de la couleur d'un élément par pourcentages

Votre objectif est cette fois de définir plusieurs étapes supplémentaires dans votre animation.

Comment faire

On est pas obligé d'utiliser forcément from et to pour créer une animation.

Pour une animation plus complexe, on va privilégier l'utilisation de pourcentages.

0% sera le départ de l'animation et 100% l'arrivée finale.

Vous allez devoir modifier le CSS pour faire changer la couleur de fond de la div en passant par 5 étapes succesives :

  • red
  • orange
  • yellow
  • green
  • blue

Etape 1 : création de l'animation

créer une règle @keyframes en prenant l'exemple suivant : 

            
            @keyframes nom_de_votre_animation {
                0% {
                    /* Vos règles CSS de départ */
                }
                25% {
                    /* Vos règles CSS à 25% de l'animation */
                }
                /* ... jusqu'à 100% */
            }

Etape 2 : Attacher l'animation à la div #animated_box

Une fois définie votre règle CSS d'animation, c'est exactement comme précédemment, il suffit d'appliquer cette animation à votre div avec les propriétés animation-name & animation-duration.
